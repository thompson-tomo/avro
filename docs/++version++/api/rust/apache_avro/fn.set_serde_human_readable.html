Function set_serde_human_readable

Source 
pub fn set_serde_human_readable(human_readable: bool)
Expand description

Set whether serializing/deserializing is marked as human readable in serde traits. This will adjust the return value of is_human_readable() for both. Once called, the value cannot be changed.

NOTE This function must be called before serializing/deserializing any data. The library leverages std::sync::Once to set the limit either when calling this method, or when decoding for the first time.
